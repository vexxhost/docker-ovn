# `docker-ovn`

This project provides builds for OVN which are being used by
[Atmosphere](https://github.com/vexxhost/atmosphere) but can be used by any
other project.

The images are published to `quay.io/vexxhost/ovn-host` (which contains
the `ovn-controller`) and `quay.io/vexxhost/ovn-central` (which contains
`ovn-northd`, `ovsdb-server` and control plane components).
